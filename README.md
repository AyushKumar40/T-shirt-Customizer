# 👕 T-Shirt Customizer

A modern and interactive 3D T-Shirt Customizer built with React, Three.js, React Three Fiber, Tailwind CSS, and Framer Motion. This application allows users to customize a T-shirt in real time by changing colors, uploading logos/images, and visualizing designs on a realistic 3D model directly in the browser.

## 🚀 Live Features

✨ Real-time 3D T-shirt visualization  
🎨 Dynamic color customization  
🖼️ Upload custom logos and images  
🔄 Interactive 3D model rotation and viewing angles  
⚡ Smooth animations and transitions  
📱 Responsive design for desktop and mobile devices  
🧠 Modern state management for seamless updates  
🎯 Fast development and optimized build using Vite

## ⚙️ Installation  
1. Clone the Repository  
    git clone https://github.com/AyushKumar40/T-shirt-Customizer.git  
2. Navigate to Project Directory
    cd T-shirt-Customizer
3. Install Dependencies
    npm install
4. Start Development Server
    npm run dev
5. Open in Browser
    http://localhost:5173
**🎯 How It Works**
 -Select a T-shirt color using the color picker.
 -Upload your own logo or image.
 -The design is automatically applied to the 3D model.
 -Rotate and inspect the T-shirt from different angles.
 -Customize until you achieve your desired design.
